A simple v2ray iOS client.

Features

1. support configure your shadowsocks server information.

2. support configure your socks/socks5 server information.

3. support configure your vmess/trojan server information.

4. support detect and insert server nodes if you copy valid JSON format of v2ray client configuration.

5. support scan QR code (generated by ss or vmess protocol) to add new node.
